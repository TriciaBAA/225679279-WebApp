Introduction to CSS Flexbox

CSS Flexible Box Layout, commonly known as Flexbox, is a layout model designed for the arrangement of elements in a responsive and efficient way in one dimension, either horizontally or vertically.

Flexbox provides an efficient way to layout, align, and distribute space among items in a container even when their size is unknown or dynamic. It is particularly useful in creating complex layouts and aligning elements within a container.

Flex Container

In Flexbox, the container is referred to as the flex container, and its direct children as flex items. To enable Flexbox layout for an element, you need to set its display property to flex or inline-flex.

For example, to create a flex container, you can use the following CSS:

.flex-container { display: flex; }
Flex Items

Flex items are the immediate children of a flex container. They are laid out along the main axis of the flex container. By default, flex items will try to fit into a single line, but they can also wrap onto multiple lines if needed.

To define an element as a flex item, you don't need to use any additional CSS. Any direct child of a flex container becomes a flex item automatically.

Flex Direction

Flex direction determines the primary axis of the flex container, along which flex items are laid out. It can be set to row, row-reverse, column, or column-reverse.

For example, to create a flex container with a column layout, you can use the following CSS:

.flex-container { display: flex; flex-direction: column; }
Justify Content

Justify content defines how flex items are aligned along the main axis of the flex container. It can be used to distribute space between and around flex items.

Common values for justify content include flex-start, flex-end, center, space-between, and space-around.

For example, to justify content to the center within a flex container, you can use the following CSS:

.flex-container { display: flex; justify-content: center; }
Align Items

Align items defines how flex items are aligned along the cross axis of the flex container. It can be used to align items at the start, end, center, or stretch them to fill the container.

Common values for align items include flex-start, flex-end, center, baseline, and stretch.

For example, to align items to the center vertically within a flex container, you can use the following CSS:

.flex-container { display: flex; align-items: center; }
Align Content

Align content defines how multiple lines of flex items are aligned along the cross axis of the flex container. It only has effect when flex items wrap onto multiple lines.

Common values for align content include flex-start, flex-end, center
